# feishuAgent

飞书校园 AI 挑战赛项目仓库，当前项目方向为 `MeetFlow - 飞书会议知识闭环 Agent`。

## 目录结构

- `config/`：配置文件目录，存放本地配置模板、环境变量说明和调度参数
- `adapters/`：平台接入层，封装飞书 API、事件适配和资源读取能力
- `workflows/`：业务工作流，承载会前、会后、风险巡检等主流程
- `tools/`：Agent 可调用工具，负责召回、抽取、摘要、任务写入等能力
- `storage/`：本地存储目录，保存 SQLite、JSON、JSONL 等运行数据
- `cards/`：飞书消息卡片模板，包含会前卡片、会后卡片、风险提醒卡片
- `scripts/`：辅助脚本目录，用于本地调试、定时触发、数据初始化和演示脚本
- `tests/`：测试目录，存放单元测试、流程测试和样例数据测试
